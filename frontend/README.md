# Hummor (Flutter)

Bootstrap this folder as a full Flutter app (Android, iOS, web, desktop) from the repository root:

```bash
cd frontend
flutter create .
flutter pub get
flutter run
```

The first command adds the missing platform folders (`android/`, `ios/`, `web/`, etc.). If your Flutter version overwrites `lib/main.dart`, restore it from git so the app stays on the Hello World screen.
